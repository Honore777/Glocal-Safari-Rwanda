# Image Storage

## Folder Structure
- `safaris/` - Safari main images and day photos
- `blogs/` - Blog post images
- `destinations/` - Destination images

## Usage
For development, place images in these folders and reference them as:
- `/images/safaris/filename.jpg`
- `/images/blogs/filename.jpg`
- `/images/destinations/filename.jpg`

## Production
For production, use cloud storage (AWS S3, Cloudinary) with CDN for faster loading.
